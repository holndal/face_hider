# face hider
![demo](https://raw.githubusercontent.com/holndal/face_hider/main/out.gif)
## Requirement
- python 3.8
- opencv
- pyvirtualcam
- OBS Studio

## Installation
download OBS from here(https://obsproject.com/ja/download)
```
pip install -r requirements.txt
```

## Usage
```
python run.py
```
and, set camera device "OBS Virtual Camera"
